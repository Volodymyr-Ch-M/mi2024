Тема 5 · Заняття 4

Підбір візуальних елементів для візуалізації даних

Дві навчальні сторінки: Питання 1 — класифікація елементів, Питання 2 — підбір під задачу. Висновки містять чекліст.

Почати з Питання 1 Структура заняття

Що всередині

  • Питання 1. Класифікація: порівняння, динаміка, структура, розподіл, простір.
  • Питання 2. Підбір: кейси «вдало/невдало», карти, кореляції, дрібні множини.
  • Висновки. Чеклісти готовності та мікропрактика.

Відкрити на весь екран

Крок 1

Вступ

Контекст заняття для військових аналітиків, мотивація та маршрут.

Крок 2

Питання 1 · Класифікація

Типи візуалізацій і що вони відповідають; приклади «вдало/невдало».

Крок 3

Питання 2 · Підбір

Як обрати форму під задачу, карти й кореляції, додаткові приклади.

Контекст

Для кого модуль

Групове заняття для військових аналітиків. Показуємо різні візуальні мови на одних даних, щоб уникати помилок вибору.

Порівняння Динаміка Структура Розподіл Простір

Навіщо модуль?

Три сторінки допоможуть структурувати заняття: вступ задає мотивацію, блоки питань показують практику й пастки, а висновки дають правила для самоперевірки.
